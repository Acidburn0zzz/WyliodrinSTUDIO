Follow the instructions here and reach a Linux environment within the router.

The link contains all the necessary informations to get confortable with Cisco's IOS environment before you reach the Linux console.

In the Linux environment, use wget and download the bundle pack for Wyliodrin server as a zip with the following command:

wget https://github.com/Wyliodrin/cisco829/zipball/master

Unzip, enter the folder and run the setup installer with the following commands:

chmod +x setup.sh
./setup.sh
